Properties
Name Description
webUrl the canonical URL on TripAdvisor for this location
phone the phone number of the location
numReviews the total number of user reviews for this location
name the name of this location
locationId the TripAdvisor ID of the location
locationString the geographical name of the location where this location resides
latitude the latitude of this location in degrees
longitude the longitude of this location in degrees
rating the rating of this location (1-5)
APIDetailUrl an API URL which will return details about this location (URL contains escaped JSON)
thumbnail a URL to a thumbnail image of the hotel
thumbnailIsBlessed Boolean whether or not the thumbnail image is blessed, reviewed at TripAdvisor as being a photo of exceptional quality
category type of property: attraction, restaurant, hotel
categoryTranslated type of property: attraction, restaurant, hotel, translated to requested language
subcategory Specific type of property, translated when possible to the requested language
subcategoryKey Key representing the specific type of property
ranking popularity of location out of all in the city
rawRanking the raw popularity value of the location
attractionTypes comma seperated list of type names describing this location
Options
Name Description
limit number of items returned
offset distance from the starting point of the list
currency currency code to use for request and response (should follow ISO 4217)
sort Method to sort the locations. Options are "popularity" and "distance". If sorting by distance, the "loc" param must be supplied
loc Coordinates of the reference point to use when sorting by distance. Must be supplied using the following comma-separated format: longitude,latitude
subcategory Filters results based on the subcategory of accommodation. Valid subcategories are: hotel, bb, inn, motel, specialty, all (shows all types of accommodations).
subcategory Filters based on the subcategory of the attraction. Valid subcategories are: bars, clubs, shopping, food_drink, ranch_farm, adventure, gear_rentals, wellness_spas, classes, sightseeing_tours, performances, sports, outdoors, amusement, landmarks, zoos_aquariums, museums, cultural, other, activities, nightlife
