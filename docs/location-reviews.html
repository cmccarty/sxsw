Properties
Name Description
id A unique ID of this review
memberId The ID of the member who wrote this review
locationId The ID of the location this review was written for (do we need this?)
rating The ID of the member who wrote this review
title The title of this review
publishedDate The date the review was published to the site
lang The language the review was written in (?)
text The full text of the review
url Direct link to this review on TripAdvisor's website
Options
Name Description
limit number of items returned
offset distance from the starting point of the list
sort The sort order to apply. One of "date", the "rating" the user gave, or the "popularity" of the review
order "asc" or "desc"
textMode How much of the review to return: "full", "snippet", or "none"
